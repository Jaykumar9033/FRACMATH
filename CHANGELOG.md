# Changelog

All notable repository changes for FRACMATH are recorded here.

## Unreleased

- Added the JOSS manuscript package under `paper/`.
- Added reviewer-facing contribution guidance in `CONTRIBUTING.md`.
- Added smoke-check instructions and `tests/run_smoke_checks.m`.
- Added a GitHub Actions workflow to build a draft JOSS PDF when paper files change.

## 1.0.0 - 2026-05-31

- Initial public software archive for the FRACMATH JOSS submission.
- Included the 2D three-point bending MATLAB benchmark and Abaqus/UMAT comparison workflow.
- Included the 3D Nooru-Mohamed benchmark.
- Included the 3D notched beam torsion benchmark.
- Included the theory manual, reproducibility guide, citation metadata, and MIT license.
